# -Revenue_analyzer_by_state_and_city
# README: Анализатор выручки по штатам и городам
## Описание проекта

Веб-приложение для анализа данных о выручке по штатам и городам на основе CSV-файлов. Позволяет загружать данные о заказах, выбирать год для анализа и получать детальную статистику по выручке.
## Основные функции
- **Загрузка CSV-файлов** с данными о заказах
- **Анализ выручки** по штатам и городам за выбранный год
- **Топ-10 рейтинги** штатов или городов по выручке
- **Статистика качества данных** с проверкой на корректность
- **Годовая статистика** с сравнением выручки по годам
- **Система предупреждений** о проблемах в данных

## Требования к данным

### Обязательные поля в CSV-файле:
- `name` - название заказа
- `segment` - сегмент
- `state` - штат
- `city` - город
- `order_date` - дата заказа (формат: ГГГГ-ММ-ДД)
- `ship_mode` - способ доставки
- `sales` - сумма продажи (числовое значение)

### Поддерживаемые форматы:
- Файлы с расширением `.csv`
- Кодировка: UTF-8
- Разделитель: запятая

## Как использовать

### 1. Загрузка данных
- Нажмите "Выберите файл" в разделе "Загрузка данных"
- Выберите CSV-файл с корректной структурой

### 2. Выбор параметров анализа
- После загрузки файла выберите год из выпадающего списка
- Выберите тип анализа: "Топ-10 штатов по выручке" или "Топ-10 городов по выручке"

### 3. Запуск анализа
- Нажмите кнопку "Проанализировать данные"
- Дождитесь обработки (отображается индикатор загрузки)

### 4. Просмотр результатов

#### Вкладка "Результаты"
- Таблица с топ-10 выбранных объектов (штатов или городов)
- Отображение выручки и доли от общей суммы

#### Вкладка "Статистика"
- Общая выручка за выбранный год
- Выручка и доля топ-10
- Количество проанализированных записей
- Сравнительная статистика по годам

#### Вкладка "Предупреждения"
- Информация о проблемах с качеством данных
- Рекомендации по улучшению данных

## Особенности обработки данных

### Проверка качества данных:
- **Корректность дат** - проверка формата ГГГГ-ММ-ДД
- **Пропущенные значения** - идентификация пустых полей
- **Выбросы** - исключение значений меньше 0 или больше 10,000,000
- **Валидация числовых полей** - проверка корректности суммы продаж

### Статистика качества отображает:
- Общее количество строк в файле
- Строки с некорректными датами
- Строки с пропущенными значениями
- Строки с выбросами
- Валидные строки после очистки

## Технические особенности

- **Адаптивный дизайн** - работает на компьютерах и мобильных устройствах

- **Визуализация данных** - таблицы с сортировкой и форматированием

- Firefox
- Safari

## Ограничения

- Рекомендуется использовать файлы до 10 МБ для оптимальной производительности
- Только CSV-формат с разделителем-запятой

## Безопасность

- Все данные обрабатываются локально в браузере пользователя
- Файлы не передаются на сервер
- Конфиденциальность данных обеспечивается
